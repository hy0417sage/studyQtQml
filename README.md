### QT가 제공하는 가치
- 크로스 플랫폼
- 확장성
- API 문서
- 유지보수성, 안정성, 호환성
- 광범위한 개발자 생태계
</br>

### Qt 지원 플랫폼
- 모든 주요 데스크탑과 임베디드 플랫폼이 지원된다.
- QT 플랫폼 추상화를 통해 QT이식 가능
</br>

### QT Quick
- QML : 사용자 인터페이스를 위한 마크업 언어
- 자바스크립트 : 동적 스크립트 언어
- QT C++ : 높은 이식성을 가진 향상된 C++ 라이브러리
</br>
